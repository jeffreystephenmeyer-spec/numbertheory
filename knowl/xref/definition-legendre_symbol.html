Definition 5.2.1. Legendre Symbol.

Given an odd prime \(p\) and an integer \(a\text{,}\) the Legendre symbol \(\left(\frac{a}{p}\right)\) is defined as \(\legendre{a}{p} = \begin{cases} 1 \amp \text{if } a \text{ is a quadratic residue modulo } p \\ -1 \amp \text{if } a \text{ is a quadratic non-residue modulo } p\\ 0 \amp \text{if } a \equiv 0 \pmod{p} \end{cases}\)
