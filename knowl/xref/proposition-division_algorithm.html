Proposition 2.1.6. The Division Algorithm.

If \(a,b\in \Z\) where \(b > 0\text{,}\) then there exists unique integers \(q,r\) satisfying
  1. \(0\le r < b\) and

Hint.

Try a proof by cases for \(a\text{.}\) If \(0\le a \le b \) then we are done (why???). So then assume \(a>b \text{.}\) Consider the descending sequence
\begin{equation*} S = \{a-bk \mid k \in \Z \text{ and } a-bk \le 0 \}. \end{equation*}
The set \(S\) is nonempty (why???). The set \(S\) has a minimal element (why???). Call this minimal element \(r\text{.}\) Show that this \(r\) has the desired properties. Why must \(0\le r < b\text{?}\) Why must it be unique? Lastly, do the case where \(a<0<b\text{.}\) What can you say about \(-a\text{?}\) How can you use the previous case to find the desired \(q\) and \(r\text{?}\) You may find Definition 1.2.2, Definition 1.3.1, and Proposition 1.2.3 useful.
