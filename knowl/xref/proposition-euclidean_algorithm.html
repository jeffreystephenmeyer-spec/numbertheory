Proposition 2.2.4. The Euclidean Algorithm.

If \(a,b\in \Z\) where \(b > 0\text{,}\) then using the division algorithm repeatedly:
\begin{align*} a \amp= bq_1 + r_1 \amp\amp\text{where } 0 \le r_1 < b,\\ b \amp= r_1 q_2 + r_2 \amp\amp\text{where }0 \le r_2 < r_1,\\ r_1 \amp= r_2 q_3 + r_3 \amp\amp\text{where }0 \le r_3 < r_2,\\ \amp\vdots \amp\amp\\ r_{n-2} \amp= r_{n-1} q_{n-1} + r_n \amp\amp\text{where } 0 \le r_n < r_{n-1},\\ r_{n-1} \amp= r_n q_n. \amp\amp \end{align*}
Let \(r_{n}\) be the last nonzero remainder. Then \(\gcd(a,b) = r_n\text{.}\)
