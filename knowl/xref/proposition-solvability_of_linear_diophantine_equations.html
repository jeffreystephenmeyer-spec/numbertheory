Proposition 2.3.2. Solvability of Linear Diophantine Equations.

Given integers \(a\) and \(b\text{,}\) the linear Diophantine equation \(ax + by = c\) has a solution if and only if \(\gcd(a,b) \mid c\text{.}\)

Hint.

Try a direct proof. For the forward direction, assume that there is a solution \((x,y)\) to the equation. Show that \(\gcd(a,b) \mid c\text{.}\) For the reverse direction, assume that \(\gcd(a,b) \mid c\text{.}\) Show that there is a solution to the equation.
