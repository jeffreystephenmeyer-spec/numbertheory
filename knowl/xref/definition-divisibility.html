Definition 2.1.1. Divisibility.

An integer \(a \) is divisible by an integer \(b\) if there exists an integer \(c\) so that \(a=bc\text{.}\)
🔗
If \(a \) is divisible by \(b\text{,}\) we say also say \(b\) divides \(a\text{.}\) We symbolize this by \(b\mid a\text{.}\) We say that \(b\) is a divisor of \(a\text{.}\)
🔗
If \(b\) does not divide \(a\text{,}\) then we write \(b\nmid a\text{.}\)
