Definition 1.2.1. Commutative Ring with Identity.

A commutative ring with identity is a set \(R\) with two binary operations:
  1. an addition operation \(+\) and a
    🔗
    🔗
  2. multiplication operation \(\cdot\)
    🔗
    🔗
satisfying 7 axioms:
(R1)
🔗
(Addition is associative.) If \(a,b,c \in R\text{,}\) then \((a+b)+c=a+(b+c)\text{.}\)
🔗
(R2)
🔗
(Addition has an identity.) There exists an element \(0 \in R\) with the property that, if \(a \in R\text{,}\) then \(0+a=a\text{.}\) We call \(0\) a zero.
🔗
(R3)
🔗
(Addition has inverses.) If \(a \in R\text{,}\) then there exists an element \(-a \in R\) so that \(a+(-a)=0\text{.}\) We call \(-a\) an additive inverse to \(a\text{.}\)
🔗
(R4)
🔗
(Addition is commutative.) If \(a,b\in R\text{,}\) then \(a+b=(b+a)\text{.}\)
🔗
(R5)
🔗
(Multiplication is associative.) If \(a,b,c \in R\text{,}\) then \((ab)c=a(bc)\text{.}\)
🔗
(R6)
🔗
(Multiplication has an identity.) There exists an element \(1 \in R\) with the property that, if \(a \in R\text{,}\) then \(1a=a\text{.}\) We call \(1\) a one.
🔗
(R7)
🔗
(Multiplication distributes across addition.) If \(a, b, c \in R\text{,}\) then \(a(b+c)=ab+ac\text{.}\)
