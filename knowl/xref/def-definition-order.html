Definition 1.3.1. Order.

A set \(X\) has a (total) order if there is a binary relationship \(\le\) satisfying:
(O1)
🔗
(reflexivity.) If \(a\in X\text{,}\) then \(a\le a \)
🔗
(O2)
🔗
(transitivity.) If \(a,b,c \in X\) satisfy \(a\le b\) and \(b\le c\text{,}\) then \(a \le c\)
🔗
(O3)
🔗
(symmetry.) If \(a,b \in X\) satisfy \(a\le b\) and \(b\le a\text{,}\) then \(a=b\)
🔗
(O4)
🔗
(totality.) If \(a,b \in X\text{,}\) then one of the following hold: \(a\le b\) or \(b\le a\)
🔗
