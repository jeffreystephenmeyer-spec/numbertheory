    Section 1.3 Well-Ordering

    In addition to the algebraic and order structure of the integers, they also possess a well-ordering.
    🔗

    Definition 1.3.1. Well-Ordered Set.

    If \(X\) is a set with a total order, it is well-ordered if every non-empty subset contains a minimal element, i.e. in symbols, if \(S\subset X\) is non-empty, then there exists an element \(a\in S\) such that, for all \(b\in S\) it is the case that \(a\le b\text{.}\)
    🔗
    🔗
    Note that to invoke well-ordering, we must certify the set in question is non-empty.
    🔗

    Proposition 1.3.2. The minimal positive element is invertible.

    If \(R\) is an ordered ring whose positive elements are well-ordered, then the minimal positive element, \(e\) is (multiplicatively) invertible.
    🔗
    🔗

    Hint.

    Try a direct proof. Let \(e\) denote the minimal positive element of \(R\text{.}\) (Why does \(e\) exist???) If \(e=1\text{,}\) we are done (why???). Otherwise \(0< e < 1\text{.}\) Look at the descending sequence
    \begin{equation*} S = \{1-ke \mid k \in \Z \text{ and } 1-ke > 0 \} \end{equation*}
    \(S\) is nonempty (why???). There exists a minimal element \(1-ne\in S\text{.}\) With minimality in mind, can \(1-ne> e\text{?}\) Can \(1-ne< e\text{?}\) Show \(1-ne=e\) (why???). Use this to find an element \(m\in \Z\) such that \(me=1\) (how???). You may find Definition 1.1.1, Definition 1.1.4, Definition 1.2.2, Definition 1.3.1 useful.
    🔗
    🔗

    Proposition 1.3.3. The minimal positive element is 1.

    If \(R\) is a ordered ring whose positive elements are well-ordered, then the minimal positive element is 1.
    🔗
    🔗

    Hint.

    Try a proof by contradiction. Let \(e\) denote the minimal positive element of \(R\) and suppose \(0 < e < 1\text{.}\) What happens when you multiply by \(e\text{?}\) Can you find a contradiction? You may find Definition 1.1.1, Definition 1.2.2, Definition 1.2.2, Proposition 1.2.4 Proposition 1.3.2 useful.
    🔗
    🔗
    Amazingly, we have shown that the structure of an ordered ring whose positive elements are well-ordered is quite rigid. There are no element between 0 and 1!
    🔗

    Proposition 1.3.4. The only positive unit is 1.

    If \(R\) is a ordered ring whose positive elements are well-ordered, then only positive unit is 1.
    🔗
    🔗

    Hint.

    Try a proof by contradiction. Let \(a\in R\) be a positive unit other than 1, Then \(1 < a\) (why???). By assumption, \(a\) is invertible, so there exists \(b \in R\) with \(ab=1\text{.}\) What happens when you multiply by \(b\text{?}\) Can you find a contradiction? You may find Definition 1.1.1, Definition 1.1.4, Proposition 1.3.3, and Proposition 1.3.2 useful.
    🔗
    🔗
    We now show that ordered rings whose positive elements are well-ordered have cancellation, a useful property we will use throughout this book.
    🔗

    Proposition 1.3.5. Products of positive elements are positive.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b \in R\) satisfy \(0 < a\) and \(0 < b\text{,}\) then \(0 < ab\text{.}\)
    🔗
    🔗

    Hint.

    Try a proof by contradiction. We know it must be the case that \(0 \le ab\) (why???) but what could go wrong and how to we show those things cannot happen? You may find Definition 1.1.1, Definition 1.2.2, Definition 1.3.1, Proposition 1.3.3 useful.
    🔗
    🔗

    Proposition 1.3.6. Well-ordered rings are domains.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b \in R\) satisfy \(ab=0\text{,}\) then either \(a = 0\) or \(b = 0\text{.}\)
    🔗
    🔗

    Hint.

    Try a proof by contrapositive. You may find Proposition 1.2.3 and Proposition 1.3.5 useful.
    🔗
    🔗

    Proposition 1.3.7. Well-ordered rings have cancellation.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b,c \in R\) satisfy \(ab=ac\text{,}\) then either \(a=0\) or \(b = c\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. Try algebraically rewriting the equation. You may find Definition 1.1.1 and Proposition 1.3.6 useful.
    🔗
    🔗
    The integers, \(\Z\text{,}\) are in fact the unique ordered commutative ring with identity whose positive elements are well-ordered. Going forward we will simply refer to this set as the integers and all properties of the integers will follow from these.
