    Section 2.3 Linear Diophantine Equations

    🔗

    Definition 2.3.1. Linear Diophantine Equation.

    A linear Diophantine equation is an equation of the form \(ax + by = c\text{,}\) where \(a\text{,}\) \(b\text{,}\) and \(c\) are integers and the solutions \(x\) and \(y\) are required to be integers.
    🔗
    🔗

    Proposition 2.3.2. Solvability of Linear Diophantine Equations.

    Given integers \(a\) and \(b\text{,}\) the linear Diophantine equation \(ax + by = c\) has a solution if and only if \(\gcd(a,b) \mid c\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. For the forward direction, assume that there is a solution \((x,y)\) to the equation. Show that \(\gcd(a,b) \mid c\text{.}\) For the reverse direction, assume that \(\gcd(a,b) \mid c\text{.}\) Show that there is a solution to the equation. You may find Definition 2.1.1, Definition 2.2.2, Proposition 2.1.5, and Proposition 2.2.5 useful.
    🔗
    🔗

    Proposition 2.3.3. Solvability of Linear Diophantine Equations (Unit Case).

    Given integers \(a\) and \(b\text{,}\) the linear Diophantine equation \(ax + by = 1\) has a solution if and only if \(\gcd(a,b) = 1\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. You may find Proposition 2.3.2 useful.
    🔗
    🔗

    Definition 2.3.4. Relatively Prime.

    Two integers \(a\) and \(b\) are relatively prime if \(\gcd(a,b) = 1\text{.}\)
    🔗
    🔗

    Proposition 2.3.5. Dividing to Get Relatively Prime.

    If \(a,b\in \Z\) and \(d = \gcd(a,b)\text{,}\) then \(a/d\) and \(b/d\) are relatively prime.
    🔗
    🔗

    Hint.

    Try a direct proof. You may find Definition 2.2.2, Proposition 2.3.2, Definition 2.1.1, Definition 1.1.1, Definition 2.3.4, and Proposition 2.3.3 useful.
    🔗
    🔗
    Given a particular solution to a linear Diophantine equation, we can find all solutions.
    🔗

    Proposition 2.3.6. The General Solution to a Linear Diophantine Equation.

    If \(a,b\in \Z\) are nonzero, \(c\in \Z\text{,}\) and \(x_0,y_0\in \Z\) are a particular solution to the equation \(ax + by = c\text{,}\) then all solutions are given by
    \begin{equation*} x = x_0 + \left(\frac{b}{d}\right)t, \quad y = y_0 - \left(\frac{a}{d}\right)t \end{equation*}
    where \(t\) runs through the integers and \(d = \gcd(a,b)\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. You must show two things: (1) that every solution has the form given, and (2) that every point of the given form is a solution. Start with (1). Suppose that \((x_1, y_1)\) is another solution. Then \(ax_1 + by_1 = c\text{.}\) Subtracting the original equation \(ax_0 + by_0 = c\) gives \(a(x_1 - x_0) + b(y_1 - y_0) = 0\text{.}\) This means \(a(x_1 - x_0) = -b(y_1 - y_0)\text{.}\) Since \(\gcd(a,b) = d\text{,}\) we can write \(a = da'\) and \(b = db'\) for some integers \(a'\) and \(b'\text{.}\) Show that \(b' \mid (x_1 - x_0)\) (why???). Deduce there exists an integer \(t\) such that \(x_1 - x_0 = tb'\text{,}\) and hence \(y_1 - y_0 = -ta'\text{.}\)
    🔗
    Now show (2). Let \(t\in \Z\text{.}\) Show that \(x_1 = x_0 + \left(\frac{b}{d}\right)t\) and \(y = y_0 - \left(\frac{a}{d}\right)t\) is a solution. Analyze \(ax_1 + by_1\text{.}\)
    🔗
    You may find Definition 2.3.1, Definition 1.1.1, Definition 2.3.4, Definition 2.2.2, Definition 2.1.1 and Proposition 1.3.7 useful.
    🔗
