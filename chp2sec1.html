    Section 2.1 The Division Algorithm

    🔗

    Definition 2.1.1. Divisibility.

    An integer \(a \) is divisible by an integer \(b\) if there exists an integer \(c\) so that \(a=bc\text{.}\)
    🔗
    🔗
    If \(a \) is divisible by \(b\text{,}\) we say also say \(b\) divides \(a\text{.}\) We symbolize this by \(b\mid b\text{.}\) If \(b\) does not divide \(a\text{,}\) then we write \(b\nmid b\text{.}\) We say that \(b\) is a divisor of \(a\text{.}\)
    🔗

    Proposition 2.1.3. Divisibility is transitive.

    If \(a,b,c\in \Z\) where \(a\mid b\) and \(b\mid c\text{,}\) then \(a\mid c\text{.}\)
