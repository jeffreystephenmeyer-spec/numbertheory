    Section 5.1 Quadratic Residues

    🔗

    Definition 5.1.1. Quadratic Residues.

    If \(p\) is an odd prime and \(a\) relative prime to \(p\text{,}\) then \(a\) is a quadratic residue modulo \(p\) if there exists an integer \(x\) such that \(x^2 \equiv a \pmod{p}\) In this case, we also say that \(a\) is a square modulo \(p\). Otherwise, \(a\) is a quadratic non-residue modulo \(p\).
    🔗
    🔗

    Proposition 5.1.2. Negative one is a quadratic residue modulo \(p\) when \(p \equiv 1 \pmod{4}\).

    If \(p\) is an odd prime, then \(-1\) is a quadratic residue modulo \(p\) if and only if \(p \equiv 1 \pmod{4}\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. First show the backward direction. Then \(p=4k+1\) (why???). Write \((p-1)!=(1 \cdot 2 \cdots (2k))\cdot((p-(2k)) \cdots (p-1))\text{.}\) Deduce \((p-1)! \equiv (2k)!^2 \pmod{p}\text{.}\) Now show the forward direction. If \(-1\) is a quadratic residue modulo \(p\text{,}\) then there exists an integer \(x\) such that \(x^2 \equiv -1 \pmod{p}\text{.}\) Raise both sides to the power of \((p-1)/2\text{.}\) Show that \((-1)^{(p-1)/2} \equiv 1 \pmod{p}\) and deduce that \(4 \mid p-1\text{.}\) You may find Definition 5.1.1, Definition 2.4.1, Proposition 4.3.3, Proposition 4.1.3, and Proposition 2.4.3 useful.
    🔗
    🔗

    Proposition 5.1.3. Euler’s Criterion.

    If \(p\) is an odd prime and \(\gcd(a,p)=1\text{,}\) then \(a\) is a quadratic residue modulo \(p\) if and only if \(a^{(p-1)/2} \equiv 1 \pmod{p}\text{.}\)
    🔗
    🔗

    Hint.

    First show the forward direction. Try a direct proof. Suppose \(a\) is a quadratic residue modulo \(p\text{.}\) Then there exists an integer \(x\) such that \(x^2 \equiv a \pmod{p}\text{.}\) Raise both sides to the power of \((p-1)/2\) and interpret. Now show the backward direction. Try a proof by contrapositive. Suppose \(a\) is not a quadratic residue modulo \(p\text{.}\) For each nonzero least residue mod \(p\text{,}\) \(r\text{,}\) there exists a unique and distinct least residue \(r'\) such that \(rr' \equiv a \pmod{p}\) (why???). Using these pairs, compute the factorial \((p-1)!\) in two ways. You may find Definition 5.1.1, Definition 2.4.1, Proposition 4.1.3, Proposition 2.4.3, Proposition 4.1.1, and Proposition 4.3.3useful.
    🔗
    🔗

    Definition 5.1.4. Centering Map.

    Fix an odd prime \(p\) and an integer \(a\) relatively prime to \(p\text{.}\) Let \(0<r\le \frac{p-1}{2}\) be a least residue modulo \(p\text{.}\) Let \(r_a\) denote the least residue of \(ar\) modulo \(p\text{.}\) Either \(r_a\le \frac{p-1}{2}\) or it is not. If it is not, then \(p-r_a\) is a least residue less than or equal to \(\frac{p-1}{2}\) (why???). We define the centering map modulo \(p\) to be the self-map \(C_a:\{1, 2, \ldots, \frac{p-1}{2}\} \to \{1, 2, \ldots, \frac{p-1}{2}\}\) defined by
    \begin{equation*} C_a(r) = \begin{cases} r_a \amp \text{if } r_a \leq \frac{p-1}{2} \\ p - r_a \amp \text{if } r_a > \frac{p-1}{2} \end{cases}. \end{equation*}
    🔗
    🔗

    Proposition 5.1.5. Centering Map is a Bijection.

    The centering map \(C_a\) is a bijection.
    🔗
    🔗

    Hint.

    First show this map is one-to-one and then deduce it is onto. Suppose \(r_1\) and \(r_2\) are least resides modulo \(p\) such that \(C_a(r_1) = C_a(r_2)\text{.}\) Show by three cases that \(r_1 = r_2\text{.}\) You may find Definition 5.1.4, Proposition 2.4.5, Proposition 2.4.6, Proposition 2.4.3, and Proposition 4.1.1 useful.
    🔗
    🔗

    Definition 5.1.6. Flip count.

    If \(p\) is an odd prime and \(a\) relatively prime to \(p\text{,}\) then define \(\beta(a,p)\) to be the number of elements in the set \(\{ar \mod p \mid r = 1, 2, \ldots, \frac{p-1}{2}\}\) that are greater than \(\frac{p-1}{2}\text{.}\) We will say \(\beta(a,p)\) is the flip count of \(a\) modulo \(p\text{.}\)
    🔗
    🔗

    Proposition 5.1.7. Gauss’s Lemma.

    If \(p\) is an odd prime and \(\gcd(a,p)=1\text{,}\) then \(a\) is a quadratic residue modulo \(p\) if and only if \(\beta(a,p)\) is even.
    🔗
    🔗

    Hint.

    Try a direct proof. Compute the product modulo \(p\) of all elements in the set \(\{ar \mod p \mid r = 1, 2, \ldots, \frac{p-1}{2}\}\) in two ways. Show \((-1)^{\beta(a,p)}=a^{(p-1)/2}\text{.}\) You may find Definition 5.1.1, Definition 2.4.1, Definition 5.1.6, Proposition 5.1.3, Proposition 2.4.3, Proposition 4.1.1, and Proposition 2.4.6useful.
    🔗
    🔗

    Proposition 5.1.8. Two is a quadratic residue modulo \(p\) when \(p \equiv \pm 1 \pmod{8}\).

    If \(p\) is an odd prime, then \(2\) is a quadratic residue modulo \(p\) if and only if \(p \equiv 1,7 \pmod{8}\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. Think about the possible cardinalities of the set \(\{2,4, \ldots, p-1\}\text{.}\) You may find Definition 5.1.1, Definition 2.4.1, and Proposition 5.1.7 useful.
