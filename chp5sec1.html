    Section 5.1 Quadratic Residues

    🔗

    Definition 5.1.1. Quadratic Residues.

    If \(p\) is an odd prime and \(a\) relative prime to \(p\text{,}\) then \(a\) is a quadratic residue modulo \(p\) if there exists an integer \(x\) such that \(x^2 \equiv a \pmod{p}\) In this case, we also say that \(a\) is a square modulo \(p\). Otherwise, \(a\) is a quadratic non-residue modulo \(p\).
    🔗
    🔗

    Proposition 5.1.2. Negative one is a square modulo \(p\) when \(p \equiv 1 \pmod{4}\).

    If \(p\) is an odd prime, then \(-1\) is a square modulo \(p\) if and only if \(p \equiv 1 \pmod{4}\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. First show the backward direction. Then \(p=4k+1\) (why???). Write \((p-1)!=(1 \cdot 2 \cdots (2k))\cdot((p-(2k)) \cdots (p-1))\text{.}\) Deduce \((p-1)! \equiv (2k)!^2 \pmod{p}\text{.}\) Now show the forward direction. If \(-1\) is a square modulo \(p\text{,}\) then there exists an integer \(x\) such that \(x^2 \equiv -1 \pmod{p}\text{.}\) Raise both sides to the power of \((p-1)/2\text{.}\) Show that \((-1)^{(p-1)/2} \equiv 1 \pmod{p}\) and deduce that \(4 \mid p-1\text{.}\) You may find Definition 2.4.1, Proposition 4.3.3, Proposition 4.1.3 useful.
