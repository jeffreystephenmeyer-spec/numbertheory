    Section 1.2 The order of the integers

    In addition to the algebraic structure of the integers, they also possess an ordering. This ordering is compatible with the algebraic operations and gives us a way to compare integers.
    🔗

    Definition 1.2.1. Order.

    A set \(X\) has a total order if there is a binary relationship \(\le\) satisfying:
    (O1)
    🔗
    (Reflexivity.) If \(a\in X\text{,}\) then \(a\le a \text{.}\)
    🔗
    (O2)
    🔗
    (Transitivity.) If \(a,b,c \in X\) satisfy \(a\le b\) and \(b\le c\text{,}\) then \(a \le c\text{.}\)
    🔗
    (O3)
    🔗
    (Antisymmetry.) If \(a,b \in X\) satisfy \(a\le b\) and \(b\le a\text{,}\) then \(a=b\text{.}\)
    🔗
    (O4)
    🔗
    (Totality.) If \(a,b \in X\text{,}\) then either \(a\le b\) or \(b\le a\text{.}\)
    🔗
    If \(a,b\in X\text{,}\) where \(a\le b\) and \(a \ne b\text{,}\) then we write \(a < b\text{.}\) The expressions \(a \le b\) and \(b \ge a\) are equivalent.
    🔗
    🔗
    Observe the symbols \(\le\text{,}\) \(\ge\text{,}\) \(<\text{,}\) and \(>\) will all be used in the usual ways.
    🔗
    A ring \(R\) is an ordered ring if \(R\) has a total order \(\le \) that is compatible with its operations as made precise in the following definition.
    🔗

    Definition 1.2.2. Ordered Ring.

    A ring \(R\) is an ordered ring if \(R\) has a total order \(\le \) satisfying:
    (OR1)
    🔗
    (Addition preserves order.) If \(a,b,c \in R\) and \(a\le b\text{,}\) then \(a +c\le b+c\)
    🔗
    (OR2)
    🔗
    (Positive scaling preserves order.) If \(a,b \in R\) satisfy \(0 \le a\) and \(0 \le b\text{,}\) then \(0\le ab\)
    🔗
    🔗
    🔗
    We call elements of an ordered ring positive if it is greater than zero and negative if it is less than zero.
    🔗
    We will establish some basic properties of ordered rings in the following propositions.
    🔗

    Proposition 1.2.3. Additive inverses of positive elements are negative.

    If \(R\) is an ordered ring and \(a \in R \) is positive (resp. negative), then \(-a\) is negative (resp. positive).
    🔗
    🔗

    Hint.

    Try a proof by cases. For each case, do a direct proof. In the first case, suppose \(0 < a \text{.}\) Show \(-a < 0 \) (why???). Now do the other case. You may find Definition 1.1.1 and Definition 1.2.2 useful.
    🔗
    🔗

    Proposition 1.2.4. Multiplication by a unit preserves inequality.

    If \(R\) is a ordered ring and \(a,b,c\in R\) are positive such that \(a\) is a unit and \(0<b<c\text{,}\) then \(0<ab<ac\text{.}\)
    🔗
    🔗

    Hint.

    Try a proof by contradiction. We know it must be the case that \(0 \le ab \le ac\) (why???) but what could go wrong and how to we show those things cannot happen? Suppose one of the undesirable things happened - how does the invertibility of \(a\) lead to a contradiction? You may find Definition 1.1.1, Definition 1.1.4, Definition 1.2.2, Definition 1.2.5 useful.
    🔗
    🔗

    Definition 1.2.5. Well-Ordered Set.

    If \(X\) is a set with a total order, it is well-ordered if every non-empty subset contains a minimal element, i.e. in symbols, if \(S\subset X\) is non-empty, then there exists an element \(a\in S\) such that, for all \(b\in S\) it is the case that \(a\le b\text{.}\)
    🔗
    🔗
    Note that to invoke well-ordering, we must certify the set in question is non-empty.
    🔗

    Proposition 1.2.6. The minimal positive element is invertible.

    If \(R\) is an ordered ring whose positive elements are well-ordered, then the minimal positive element, \(e\) is (multiplicatively) invertible.
    🔗
    🔗

    Hint.

    Try a direct proof. Let \(e\) denote the minimal positive element of \(R\text{.}\) (Why does \(e\) exist???) If \(e=1\text{,}\) we are done (why???). Otherwise \(0< e < 1\text{.}\) Look at the set
    \begin{equation*} S = \{1-ke >0 \mid k \in \Z \} \end{equation*}
    \(S\) is nonempty (why???). There exists a minimal element \(1-ne\in S\text{.}\) With minimality in mind, can \(1-ne> e\text{?}\) Can \(1-ne< e\text{?}\) Show \(1-ne=e\) (why???). Use this to find an element \(m\in \Z\) such that \(me=1\) (how???). You may find Definition 1.1.1, Definition 1.1.4, Definition 1.2.2, Definition 1.2.5 useful.
    🔗
    🔗

    Proposition 1.2.7. The minimal positive element is 1.

    If \(R\) is a ordered ring whose positive elements are well-ordered, then the minimal positive element is 1.
    🔗
    🔗

    Hint.

    Try a proof by contradiction. Let \(e\) denote the minimal positive element of \(R\) and suppose \(0 < e < 1\text{.}\) What happens when you multiply by \(e\text{?}\) Can you find a contradiction? You may find Definition 1.1.1, Definition 1.2.2, Definition 1.2.2, Proposition 1.2.4 Proposition 1.2.6 useful.
    🔗
    🔗
    Amazingly, we have shown that the structure of an ordered ring whose positive elements are well-ordered is quite rigid. There are no element between 0 and 1!
    🔗

    Proposition 1.2.8. The only positive unit is 1.

    If \(R\) is a ordered ring whose positive elements are well-ordered, then only positive unit is 1.
    🔗
    🔗

    Hint.

    Try a proof by contradiction. Let \(a\in R\) be a positive unit other than 1, Then \(1 < a\) (why???). By assumption, \(a\) is invertible, so there exists \(b \in R\) with \(ab=1\text{.}\) What happens when you multiply by \(b\text{?}\) Can you find a contradiction? You may find Definition 1.1.1, Definition 1.1.4, Proposition 1.2.7, and Proposition 1.2.6 useful.
    🔗
    🔗
    We now show that ordered rings whose positive elements are well-ordered have cancellation, a useful property we will use throughout this book.
    🔗

    Proposition 1.2.9. Products of positive elements are positive.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b \in R\) satisfy \(0 < a\) and \(0 < b\text{,}\) then \(0 < ab\text{.}\)
    🔗
    🔗

    Hint.

    Try a proof by contradiction. We know it must be the case that \(0 \le ab\) (why???) but what could go wrong and how to we show those things cannot happen? You may find Definition 1.1.1, Definition 1.2.2, Definition 1.2.5, Proposition 1.2.7 useful.
    🔗
    🔗

    Proposition 1.2.10. Well-ordered rings are domains.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b \in R\) satisfy \(ab=0\text{,}\) then either \(a = 0\) or \(b = 0\text{.}\)
    🔗
    🔗

    Hint.

    Try a proof by contrapositive. You may find Proposition 1.2.3 and Proposition 1.2.9 useful.
    🔗
    🔗

    Proposition 1.2.11. Well-ordered rings have cancellation.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b,c \in R\) satisfy \(ab=ac\text{,}\) then either \(a=0\) or \(b = c\text{.}\)
    🔗
    🔗

    Hint.

    Try a direct proof. Try algebraically rewriting the equation. You may find Definition 1.1.1 and Proposition 1.2.11 useful.
    🔗
    🔗
    The integers, \(\Z\) are in fact the unique ordered commutative ring with identity whose positive elements are well-ordered. Going forward we will simply refer to this set as the integers and all properties of the integers will follow from these.
