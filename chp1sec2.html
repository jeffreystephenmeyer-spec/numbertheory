    Section 1.3 The order of the integers

    🔗

    Definition 1.3.1. Order.

    A set \(X\) has a (total) order if there is a binary relationship \(\le\) satisfying:
    1. (reflexivity.) If \(a\in X\text{,}\) then \(a\le a \)
      🔗
      🔗
    2. (transitivity.) If \(a,b,c \in X\) satisfy \(a\le b\) and \(b\le c\text{,}\) then \(a \le c\)
      🔗
      🔗
    3. (symmetry.) If \(a,b \in X\) satisfy \(a\le b\) and \(b\le a\text{,}\) then \(a=b\)
      🔗
      🔗
    4. (totality.) If \(a,b \in X\text{,}\) then one of the following hold: \(a\le b\) or \(b\le a\)
      🔗
      🔗
    🔗
    🔗

    Definition 1.3.2. Ordered Ring.

    A ring \(R\) is an ordered ring if \(R\) has an order that is compatible with its operations in the following satisfying:
    1. (translation.) If \(a,b,c \in R\) and \(a\le b\text{,}\) then \(a +c\le b+c\)
      🔗
      🔗
    2. (scaling.) If \(a,b \in R\) satisfy \(0 \le a\) and \(0 \le b\text{,}\) then \(0\le ab\)
      🔗
      🔗
    🔗
    🔗
    Observe the symbols \(\le\text{,}\) \(\ge\text{,}\) \(<\text{,}\) and \(>\) will all be used in the usual ways.
    🔗
    We call elements of an ordered ring positive if it is greater than zero and negative if it is less than zero.
    🔗

    Proposition 1.3.3. .

    If \(R\) is an ordered ring and \(a \in R \) is positive (resp. negative), then \(-a\) is negative (resp. positive).

    Definition 1.3.4. Well-Ordered Set.

    If \(X\) is a set with a total order, it is well-ordered if every subset contains a minimal element, i.e. in symbols, if \(S\subset X\) then there exists an element \(a\in S\) such that, for all \(b\in S\) it is the case that \(a\le b\text{.}\)
    🔗
    🔗

    Proposition 1.3.5. .

    If \(R\) is a ordered ring whose positive elements are well-ordered, then the minimal positive element, \(\epsilon\) is invertible.

    Proposition 1.3.8. Well-Ordered Rings are Domains.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b \in R\) satisfy \(ab=0\text{,}\) then either \(a=0\) or \(b = 0\text{.}\)

    Proposition 1.3.9. Well-Ordered Rings Have Cancellation.

    If \(R\) is a ordered ring whose positive elements are well-ordered, and \(a,b,c \in R\) satisfy \(ab=ac\text{,}\) then either \(a=0\) or \(b = c\text{.}\)

    The integers, \(\Z\) are in fact the unique ordered commutative ring with identity whose positive elements are well-ordered. Going forward we will simply refer to this set as the integers and all properties of the integers will follow from these.
